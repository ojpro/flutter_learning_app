# Flutter App for learning

This is an app I am building while learning Flutter.


## Run Locally

Clone the project

```bash
  git clone https://github.com/ojpro/flutter_learning_app.git
```

Go to the project directory

```bash
  cd flutter_learning_app
```

Start the app

```bash
  flutter run
```
